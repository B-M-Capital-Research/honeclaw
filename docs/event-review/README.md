# Event Review Reports

Daily event-engine push quality reviews live here.

Each report should be named `YYYY-MM-DD.md` for the reviewed Asia/Shanghai local date. Reports should summarize which actors received valid pushes, whether timing and content were reasonable, and whether any useful events were filtered, omitted, capped, cooled down, or otherwise missed.

Do not commit raw runtime exports, SQLite databases, full message bodies, or full article text here. Keep reports concise and evidence-backed with event ids, local send times, statuses, titles, and URLs when needed.
