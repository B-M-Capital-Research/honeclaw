# `docs/` 目录说明

本目录用于**本地工作笔记**（计划、交接、草稿等）。除本文件外，`docs/` 下其他内容**默认不纳入 Git**（见仓库根目录 `.gitignore`）。

对外文档请以仓库根目录为准：

- 英文：`README.md`
- 中文：`README_ZH.md`
- 交互式系统架构图：`resources/architecture.html`
- 协作与 CI 约定：`AGENTS.md`、`CONTRIBUTING.md`
