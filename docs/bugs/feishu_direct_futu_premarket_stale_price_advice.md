# Bug: Feishu 直聊在 FUTU 盘前暴跌时仍用常规交易旧价给抄底区间

- **发现时间**: 2026-05-22 19:02 CST
- **Bug Type**: Business Error
- **严重等级**: P3
- **状态**: Fixed

## 证据来源

- `data/sessions.sqlite3` -> `session_messages`
  - `session_id=Actor_feishu__direct__ou_5f636d6d7c80d333e41b86ae79d07adca8`
  - `2026-05-22 16:40 CST` 用户问：`今天富途被罚，这件事对富途影响多大？什么价格可以抄底`。
  - `2026-05-22 16:41 CST` assistant final 明确使用 `FUTU 最新可校验价格 123.86 美元，跌 0.69%`，并给出 `110 美元以下才开始有较好赔率，95 到 105 美元更像深度恐慌买点`。
  - `2026-05-22 16:46 CST` 用户补充：`盘前已经到80了`。
  - `2026-05-22 16:47 CST` assistant final 承认：`我刚才的数据源还停留在常规交易口径 123.86 美元，未更新盘前`，并把判断下修为 `80 美元已经进入深度恐慌区`。
- 最近四小时窗口 `2026-05-22 15:02-19:02 CST` 共有 `9` 个 user turn 与 `9` 个 assistant final，Feishu / Web 直聊均有 assistant final 收口；assistant final 污染扫描未命中空回复、绝对路径、工具轨迹、原始 ACP 文本、compact marker、provider 原始错误、panic 或 `quota exhausted`。本缺陷是强时效投研回答的价格口径质量问题，不是直聊链路失败。

## 端到端链路

1. 用户在 Feishu direct 中询问当天监管事件对 FUTU 的影响，并要求给出可抄底价格。
2. assistant 正确识别为当天监管新闻，但行情字段仍采用常规交易旧价 / 昨收口径。
3. assistant 基于 `123.86` 美元给出 `105-110` 与 `95-105` 的分层抄底区间。
4. 用户指出盘前已到 `80` 美元附近。
5. assistant 承认上一轮数据源未更新盘前，并重新给出更低区间和小仓位建议。

## 期望效果

- 当用户问题同时包含 `今天`、监管事件、个股价格和 `抄底` 这类强时效操作语义时，行情口径必须先覆盖盘前 / 盘后 / 实时可得价格，再给买点区间。
- 如果工具只能返回常规交易收盘或延迟行情，应明确说明“未覆盖盘前实时价”，并避免把旧价当作当前决策锚。
- 若事件正在盘前快速定价，回答应先提示需以券商实时盘前价或权威实时行情为准，再给条件化框架。

## 当前实现效果

- assistant 把常规交易旧价 `123.86` 当成当前可决策价格，导致给出的抄底区间明显滞后于真实盘前价格。
- 用户需要主动补充 `盘前已经到80了`，assistant 才调整判断。

## 用户影响

- 这是质量性 bug，不是功能性 bug。
- 用户询问的是强时效投资决策，旧价锚点会显著影响“是否已经进入恐慌区 / 能否抄底”的判断。
- 本问题不影响消息投递、任务创建、会话收口、数据安全或系统稳定性；同窗直聊均正常收口，也没有错误投递或未回复，因此定级为 `P3`。

## 根因判断

- 初步判断是行情工具或回答层没有把盘前实时口径作为强时效问题的必选校验项。
- 回复虽然核验了监管新闻，但价格字段使用常规交易 snapshot 后继续给出操作区间，没有把“盘前快速下跌”作为缺失风险暴露出来。
- 该问题与 `feishu_scheduler_stale_price_fallback_after_data_fetch_failure.md` 不同：那条覆盖 scheduler 在数据抓取失败后仍发送旧价格早报；本单是 Feishu direct 单轮强时效投研回答在盘前行情口径上滞后。

## 下一步建议

- 对直聊投研 prompt 或行情工具结果消费层增加口径约束：若用户使用 `今天 / 刚刚 / 盘前 / 盘后 / 现在 / 抄底` 等强时效词，必须优先校验盘前 / 盘后 / 实时可得价，或显式声明不可得。
- 对操作建议类回复增加价格口径字段：`常规交易收盘价`、`盘前/盘后价`、`数据时间`、`是否可作为当前决策锚`。
- 后续巡检若再次出现用户用实时价纠正 assistant 旧价，可将本单升级为更系统的 direct 行情口径缺陷。
