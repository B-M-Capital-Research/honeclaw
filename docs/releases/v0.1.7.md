## 升级后会有什么变化

- Web 记忆页新增公司画像列表与详情视图，支持查看画像正文、事件时间线和追踪状态
- 从知识记忆表可以直接打开公司画像；未建档时会提示通过 agent 建立
- `hone-cli cleanup` 默认会先询问是否删除：
  - `~/.honeclaw/data`
  - `~/.honeclaw/releases`
  - `~/.honeclaw/current`
  - 以及是否额外删除 `~/.honeclaw/config.yaml` / `soul.md`
- `brew uninstall honeclaw` 仍只卸载程序本体；如果还想删用户数据，先执行 `hone-cli cleanup`

## 安装与更新

- `curl | bash` 用户：重新运行安装脚本即可切到新版本
- Homebrew 用户：`brew update && brew upgrade honeclaw`
- 卸载前如需清理本地数据：
  - 交互式：`hone-cli cleanup`
  - 全量非交互：`hone-cli cleanup --all --yes`

## 已知注意事项

- 如果你已经先执行了 `brew uninstall honeclaw`，之后又想删数据，需要手动删除 `~/.honeclaw`
