# v0.1.18

发布日期：2026-04-13

## Highlights

- 为 multi-agent search 阶段新增 actor sandbox 内的本地只读检索能力，支持列目录、关键词搜索和按行读取文本文件。
- 将本地文件检索提升为与 `web_search` / `data_fetch` 同级的搜索手段，覆盖 `company_profiles/`、`uploads/`、`runtime/` 等用户本地持久化信息。
- 保持现有多阶段流转不变：search 阶段自由决定是否用工具；一旦 search 阶段调用任意工具，就继续进入 answer 阶段。

## User-Facing Changes

- 当用户询问“以前研究过哪些公司”“本地档案里有哪些内容”这类依赖 actor 本地持久化状态的问题时，search 阶段不再默认回答“没有记忆/无法访问文件”，而是可以先检查当前 actor sandbox 内的本地文件。
- README 与 README_ZH 增补了公司画像长期沉淀能力说明。
