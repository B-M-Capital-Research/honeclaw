## 升级后会有什么变化

- 公司画像主存储改成 Markdown 文档与事件目录，agent 会优先使用 runner 原生读写能力维护 `company_profiles/`
- 上传附件仍然可以用于当次分析，但不会再自动沉淀到 KB；若要长期保留，需要由 agent 写回公司画像
- 前端只负责展示和删除画像，不再提供 KB 页面；公司数据的创建、更新与追加事件统一通过 agent 协作完成
