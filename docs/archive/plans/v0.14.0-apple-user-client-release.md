## Goal

Publish the current HONE public user experience as v0.14.0 macOS and iOS release assets, with aligned metadata, bilingual notes, reproducible verification, and checked GitHub Release artifacts.

## Scope

- Align workspace, Tauri, and iOS marketing versions to 0.14.0.
- Publish the Universal macOS DMG, iOS Simulator App, and complete Xcode project.
- Refresh release notes and architecture documentation.
- Preserve the remote-only Apple client boundary and make the macOS bundle ad-hoc signature verifiable.

- Local Universal app/DMG inspection confirmed version, architectures, minimal contents, hardened runtime, and bundle-level ad-hoc signature.
