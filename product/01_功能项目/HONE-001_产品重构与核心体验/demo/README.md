# HONE Live Demo

这是 HONE 产品重构的独立交互原型，用于产品评审和需求验证，不依赖真实行情、账户或生产服务。

## 已实现范围

- 官网介绍首页：产品定位、四个研究动作、研究闭环和设计原则。
- 投资：采用“专业持仓优先 + 单条 Agent 研究信号”方向；支持资产/持仓/表现切换、组合收益、只读账户状态、持仓同步/导入三步链路、公司 K 线、成本线、历史买入点、财务指标和研究主线。
- 洞察：类似 Twitter / 知识星球的统一图文消息流；帖子可包含图片和可选深度文章，当前仅老王及授权编辑发布，并预留未来社区讨论结构。
- Agent：研究模式、快速能力、今日线索、历史研究、结构化回答、来源和后续动作。
- 跟踪：今日时间线、事件详情、结构化结果、准备清单、整月日历、全部任务管理、任务编辑、历史和三步新建跟踪向导。
- 我的：账户、会员用量、研究偏好编辑、设备与登录、安全活动、快捷显示设置、长期记忆、隐私，以及完整订阅选档与模拟支付流程。
- 账户入口：桌面侧栏不再重复显示“我的”；左下角固定头像栏统一打开个人中心、会员、快捷设置、帮助与退出菜单。
- 响应式导航：桌面左栏先展示投资/洞察/跟踪工作台，再展示 AI 研究（新研究、搜索、最近对话、全部记录）；头像与设置固定在左下角，右上角只保留全局搜索；不设置“返回官网”和重复 Agent 入口。移动端保持五入口底栏，Agent 居中突出。

## 运行

```bash
npm install
npm run dev -- --host 0.0.0.0 --port 4173
```

打开 `http://127.0.0.1:4173/`。

## 技术边界

- React 19 + Vite 6 + Tailwind CSS 4。
- 使用实际 shadcn/Radix 组件，以黑白灰 token 构建。
- 图标使用 Phosphor Icons；组合图表使用 Recharts，公司 K 线使用 Lightweight Charts。
- 当前数据全部是产品验证用 mock data。
