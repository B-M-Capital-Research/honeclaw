🔄 AgentSession
1. Session load - restore history from disk
2. Session compression - when messages exceed 20 / 80KB, use LLM summarization and keep the latest 6 messages
3. Prompt assembly - system_prompt + current time + channel formatting guidance + privacy guard + KB summary
4. Progress events - Progress / StreamDelta / ToolStatus / Done / Error
5. Session write-back - persist messages + metadata to JSON
